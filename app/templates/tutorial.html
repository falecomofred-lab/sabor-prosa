🚀 Primeiro Acesso

1. Abra o PowerShell e digite:

cd C:\Users\Frederico\projetos\sabor_prosa_emporio_v2\backend
python -m uvicorn app.main:app --reload

2. Acesse no navegador:

http://localhost:8000/dashboard

3. No celular (mesma rede WiFi):

http://192.168.15.8:8000/dashboard

🛒 Como Usar o PDV

1. Clique em "🛒 PDV" no menu

2. Toque nos produtos mais vendidos (topo da tela)

3. Ou use o 📷 scanner para ler código de barras

4. Ou digite o nome do produto na busca

5. Clique em "💰 Finalizar"

6. Digite o valor recebido → TROCO GIGANTE na tela!

💡 Use "⏸️ Pausar Venda" para atender outro cliente.

📦 Como Cadastrar um Produto

1. Clique em "📦 Produtos"

2. Preencha: Nome, Categoria, Código de Barras

3. Preço de Custo e Venda → Margem calcula sozinha!

4. Tire uma foto com o celular 📸

5. Clique em "💾 Salvar"

🛍️ Vitrine do Cliente

1. Acesse: /vitrine

2. Envie o link para clientes via WhatsApp

3. Cliente vê produtos com fotos e preços

4. Monta carrinho e finaliza → Pedido cai no seu WhatsApp!

🚚 Fornecedores (30 segundos!)

1. Digite o CNPJ e clique em 🔍

2. Sistema preenche: nome, telefone, email, endereço!

3. Digite o CEP e clique em 📍

4. Endereço completo preenchido automaticamente!

📱 Criar Posts com IA

1. Clique em "📱 Conteúdo"

2. Escolha o tipo: Produto do Dia, Harmonização, Promoção

3. Selecione o produto

4. Clique em "🎨 Gerar Post"

5. IA cria texto com hashtags e emojis!

6. Clique em "📋 Copiar" e cole no Instagram

🗺️ Rotas de Entrega

1. Clique em "🗺️ Rotas"

2. Clique em "📋 Listar Entregas"

3. Veja os pontos no mapa

4. Clique em "🗺️ Otimizar Rota"

5. Google calcula melhor caminho, distância e tempo

💡 Dicas Para o Dia a Dia

✅ Cadastre fotos dos produtos (câmera do celular)

✅ Use o código de barras para agilizar o PDV

✅ Verifique os alertas de estoque baixo no Dashboard

✅ Use os Gatilhos para reengajar clientes

✅ Gere posts toda semana com a IA

✅ Monte kits para datas especiais

✅ Use o Checklist antes de cada feira

✅ O PDV funciona OFFLINE (sem internet)
