🚀 Primeiro Acesso

1. Abra o PowerShell (Windows) e digite:

cd C:\Users\Frederico\projetos\sabor_prosa_emporio_v2\backend
python -m uvicorn app.main:app --reload

2. Abra o navegador e acesse:

http://localhost:8000/dashboard

3. Para acessar pelo celular (mesma rede WiFi):

http://192.168.15.8:8000/dashboard

💡 O IP aparece no terminal quando o servidor inicia.

📊 Dashboard - Seu Painel de Controle

Ao abrir o sistema, você vê:

  • ✅ 3 botões rápidos: Venda Rápida, Novo Produto, Criar Post
  • ✅ Faturamento do dia: Comparativo com semana anterior
  • ✅ 4 cards: Produtos, Estoque, Margem, Alertas
  • ✅ Produtos com estoque baixo: Com botão WhatsApp para pedir reposição
  • ✅ Gráfico de margens: Barras coloridas por categoria

📦 Como Cadastrar um Produto

1. Clique em "📦 Produtos" no menu lateral

2. Preencha o formulário:

  • 📝 Nome do produto (obrigatório)
  • 🏷️ Categoria (Queijos, Vinhos, Geleias, etc.)
  • 🔢 Código de barras (para usar no PDV)
  • 💰 Preço de custo e venda (a margem calcula sozinha!)
  • 📸 Foto do produto (use a câmera do celular!)

3. Clique em "💾 Salvar Produto"

💡 Dica: Use o botão "📋 Duplicar" para criar produtos parecidos rapidamente.

🛒 Como Usar o PDV (Balcão de Vendas)

1. Clique em "🛒 PDV" no menu

2. Toque nos produtos mais vendidos (aparecem no topo)

3. OU escaneie/digite o código de barras e pressione Enter

4. O carrinho mostra todos os itens e o total

5. Clique em "💰 Finalizar"

6. Digite o valor recebido do cliente

7. O TROCO aparece GIGANTE na tela!

💡 Use "⏸️ Pausar Venda" para atender outro cliente e retomar depois.

🛍️ Vitrine do Cliente (PWA)

1. Acesse: http://localhost:8000/vitrine

2. Envie esse link para seus clientes via WhatsApp ou Instagram

3. O cliente vê os produtos com fotos, filtra por categoria

4. Monta o carrinho e finaliza o pedido

5. O pedido já chega formatado no seu WhatsApp!

💡 O cliente não precisa baixar nada - funciona direto no navegador!

🚚 Cadastro de Fornecedores (30 segundos!)

1. Clique em "🚚 Fornecedores"

2. Digite o CNPJ e clique em 🔍

3. O sistema preenche automaticamente: nome, telefone, email, endereço!

4. Digite o CEP e clique em 📍

5. O endereço é preenchido automaticamente!

💡 Tempo estimado: 30 segundos por fornecedor!

📱 Criar Posts e Vídeos com Inteligência Artificial

1. Clique em "📱 Conteúdo" ou "🎬 Estúdio"

2. Escolha o tipo de post:

  • ✨ Produto do Dia
  • 🍷 Harmonização
  • ⚠️ Promoção
  • 💡 Dica do Dia

3. Selecione o produto

4. Clique em "🎨 Gerar Post"

5. A IA cria o texto completo com hashtags e emojis!

6. Clique em "📋 Copiar" e cole no Instagram

💡 Use "🎬 Gerar Roteiro" para criar roteiros de vídeos para Reels!

🗺️ Roteirizador de Entregas

1. Clique em "🗺️ Rotas"

2. Veja o Resumo do Dia com todas as entregas

3. Clique em "🚀 Otimizar Rota"

4. O sistema reorganiza na melhor ordem

5. Use o botão "📱 Avisar" para mandar WhatsApp para cada cliente

💡 Economize gasolina e tempo seguindo a rota otimizada!

💡 Dicas Para o Dia a Dia

✅ Cadastre fotos dos produtos - use a câmera do celular!

✅ Use o código de barras para agilizar o PDV

✅ Verifique os alertas de estoque baixo no Dashboard

✅ Use o WhatsApp para pedir reposição (botão 📱)

✅ Gere posts toda semana com a IA (Estúdio de Conteúdo)

✅ Monte kits para datas especiais (Dia dos Namorados, Natal)

✅ Cadastre clientes fiéis e use os Gatilhos para reengajar

✅ Use o Checklist antes de cada feira/evento

✅ O PDV funciona OFFLINE (sem internet!)
